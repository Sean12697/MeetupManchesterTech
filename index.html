Manchester Tech Meetups

Information

This is a project that I am developing for tech enthusiasts (from students to CEO's alike) to be able to find Meetup Groups that they might be interested in joining, or to find out what events are happening in Manchester.

I have developed this with Javascript using the RESTful Meetup API alongside the Google Calendar API (to get the TechNW Calendar) to generate a calendar based on the groups you select (all selected by default), the TechNW Calendar, or a combination of the two.

If anything is unclear, there could be some improvements, something is not working (or displaying correctly), a Meetup Group that should be on here is not, or you have general feedback, please do get in touch and for now, I hope you enjoy using my service ☺
